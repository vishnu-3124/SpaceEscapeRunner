# SpaceEscapeRunner

An endless mobile runner game built with React Native and Expo where player dodge falling asteroids using responsive touch controls.

## Setup & Installation Instructions

Follow these quick steps to run the game locally on your machine and test it on your device:

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### 2. Clone the Repository
Clone this project to your local machine using git:

```Bash
git clone [https://github.com/YOUR_USERNAME/SpaceEscapeRunner.git](https://github.com/YOUR_USERNAME/SpaceEscapeRunner.git)
cd SpaceEscapeRunner
```

3. Install Dependencies
Install the required node modules and Expo libraries:

```Bash
npm install
```

4. Start the Development Server
Launch the Expo CLI engine:

```Bash
npx expo start
```

5. Test via Expo Go
Download the Expo Go app on your Android or iOS device.
Ensure both your computer and your phone are on the same Wi-Fi network.
Scan the generated terminal QR code using your phone's camera to run the app instantly!

3. Replace `YOUR_USERNAME` in the clone link with your actual GitHub username.
